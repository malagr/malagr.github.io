This is a flash video set up to run using JW FLV Media Player 4.7 and embedded using the static method of swfobject 2.2. This should load up the video even without Javascript support.

In Internet Explorer, check that you have clicked any prompt on the information bar to allow active content and that you have not disallowed ActiveX altogether (Internet options/Security/Internet/Custom Level).

Most likely the problem is an old version of Adobe Flash Player in your current browser.
To install or update click (but watch out to untick the OPTIONAL OFFERS)
Test your current AFP installation.
It sometimes helps to deinstall your old version first.

However, if you don’t have administrator access, Internet Explorer won’t let you update (but is silent about why not). In Firefox, there may be a workaround for this problem by downloading the update files manually and copying them to the necessary folders.

Otherwise the best I can do is provide access to an .avi mpg4 Xvid version of the videos; if you click on the links, your video player (e.g. Windows Media Player) should open and run it. You may need to upgrade to the latest version or download the Xvid codec.
